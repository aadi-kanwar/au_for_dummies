# AU for Dummies

## Installing VitePress
> npm add -D vitepress  

> pnpm add -D vitepress  
 
> yarn add -D vitepress  

## Starting the local server
> npm run docs:dev  

> pnpm run docs:dev  

> yarn docs:dev  

After starting the server, a localhost will be created with the following link.  
> http://localhost:5173
